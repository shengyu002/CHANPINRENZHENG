        <h1>免于符合性评价对象的器材</h1>
        <div class="header-meta">RRA KSDB 第9-4号（2024.12.23.）</div>

        <h2>□ 概要</h2>
        <p>ㅇ 关于《广播通信器材等符合性评价相关告示》［附表1］第11项中“会产生电磁干扰或受电磁影响的设备”之中，<span class="highlight">免于符合性评价的器材信息共享</span>。</p>

        <h2>□ 依据</h2>
        <p>ㅇ <span class="highlight">《广播通信器材等符合性评价相关告示》［附表1］第11项</span></p>

        <h2>□ 内容</h2>
        <p>ㅇ 根据《广播通信器材等符合性评价相关告示》［附表1］第11项规定，属于该项范围内但<span class="highlight-warn">不属于符合性评价对象（免除评价对象）</span>的器材如下：</p>

        <!-- 1. 无线电应用设备类 -->
        <h2>1. 无线电应用设备类及相关设备中的免除对象</h2>
        <ul>
            <li>根据《无线电法》第58条第1款规定，需要取得许可的无线电应用设备。</li>
        </ul>

        <!-- 2. 生活家电设备类 -->
        <h2>2. 生活家电设备类中的免除对象</h2>

        <h3>㈎ 因使用频率较低、电磁危害程度较低的以下设备</h3>
        <h4>1)</h4>
        <ul>
            <li>鸡毛脱毛机</li>
            <li>真空米桶</li>
            <li>冰淇淋／葡萄酒／烧酒分配器</li>
        </ul>

        <h4>2) 皮肤护理设备：</h4>
        <ul>
            <li>电动洁面仪</li>
            <li>喷雾仪</li>
            <li>皮脂管理仪</li>
            <li>角质管理仪</li>
            <li>皮脂及角质管理仪</li>
            <li>加热式眉毛夹</li>
            <li>扁桃体结石吸引器</li>
            <li>耳垢吸引器</li>
            <li>鼻涕吸引器</li>
            <li>UV指甲固化灯</li>
        </ul>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h4>3)</h4>
        <ul>
            <li>电动桌</li>
            <li>电动饭柜</li>
            <li>电动衣柜</li>
        </ul>

        <h4>4)</h4>
        <ul>
            <li>自动宠物喂食器</li>
            <li>宠物脚部清洗机</li>
            <li>移动式宠物笼（仅限采用USB或电池（包括充电电池）供电运行的设备）</li>
            <li>宠物排泄物处理器</li>
            <li>动物用UV杀菌刷</li>
            <li>狗用超声波／电击式防吠器</li>
        </ul>

        <h3>㈏</h3>
        <p>不使用电动机，仅短时间（<span class="highlight">2分钟以内</span>）产生光和声音，或仅产生光、声音的娱乐设备及玩具类（儿童玩具）</p>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h3>㈐</h3>
        <ul>
            <li>真空泵</li>
            <li>空气泵</li>
            <li>油泵</li>
            <li>砂泵</li>
        </ul>

        <h3>㈑ 生活家电设备类中符合以下任一条件的器材：</h3>
        <h4>1)</h4>
        <p>用于轻工业、农业、工业环境（不包括第“车项3”规定环境）的设备</p>

        <h4>2)</h4>
        <p>属于《医疗器械法》第2条第1款规定的<span class="highlight">医疗器械</span></p>

        <h4>3)</h4>
        <p>安装于非通用机械、设备上的<span class="highlight">特殊结构设备</span></p>

        <h4>4)</h4>
        <p>采用USB或电池（包括充电电池）供电运行的：</p>
        <ul>
            <li>手部消毒器等电气喷雾设备</li>
            <li>电动喷雾器</li>
            <li>电动香氛除臭设备</li>
            <li>学校等教育机构用于科学实验的组装用品套件（套装）</li>
        </ul>

        <h4>5)</h4>
        <p>额定输入功率<span class="highlight">超过10kW</span>的设备</p>

        <!-- 3. 电气/电动工具类 -->
        <h2>3. 电气／电动工具类中的免除对象</h2>

        <h3>㈎ 仅用于工业现场的设备：</h3>
        <ul>
            <li>导管弯曲机</li>
            <li>钢筋绑扎机</li>
            <li>螺栓压缩机</li>
            <li>混凝土振动器</li>
            <li>电动耕作机</li>
            <li>家畜饲料粉碎机等建筑设备类</li>
        </ul>

        <h3>㈏ 无电路结构，仅由简单发热线构成的：</h3>
        <ul>
            <li>肩部加热垫</li>
            <li>电加热设备</li>
        </ul>

        <h3>㈐ 学校等教育机构用于科学实验的组装用品套件（套装）</h3>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h3>㈑ 额定输入功率超过：</h3>
        <ul>
            <li>10kW（电焊机类超过<span class="highlight">10kVA</span>）的设备。</li>
        </ul>

        <!-- 4. 照明设备类 -->
        <h2>4. 照明设备类中的免除对象</h2>

        <h3>㈎ 学校等教育机构用于科学实验的组装用品套件（套装）</h3>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h3>㈏ 仅具有简单照明功能的设备或类似设备：</h3>
        <ul>
            <li>简单开／关功能</li>
            <li>闪烁功能</li>
            <li>简单亮度调节功能</li>
            <li>简单颜色调节功能</li>
        </ul>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <!-- 5. 多媒体设备类 -->
        <h2>5. 多媒体设备类中的免除对象</h2>

        <h3>㈎ 无主动电子电路放大器的：</h3>
        <ul>
            <li>耳机</li>
            <li>扬声器</li>
            <li>麦克风</li>
        </ul>

        <h3>㈏ 多媒体设备类中符合以下任一条件的器材：</h3>
        <h4>1)</h4>
        <p>额定输入：<span class="highlight">交流1,000V以上</span>或<span class="highlight">直流1,500V以上</span>的设备。</p>

        <h4>2)</h4>
        <p>安装于非通用机械、设备上的特殊结构设备。</p>

        <h4>3)</h4>
        <p>学校等教育机构用于科学实验的组装用品套件（套装），以及仅具有简单时钟功能的电子钟</p>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h4>4)</h4>
        <p>一次性使用且无法重复使用的设备：</p>
        <ul>
            <li>圣诞卡</li>
            <li>生日蜡烛等</li>
        </ul>

        <h4>5)</h4>
        <p>无需独立电源即可一对一连接的：</p>
        <ul>
            <li>电缆</li>
            <li>电缆连接器</li>
            <li>电缆附件</li>
        </ul>

        <!-- 6. 数字设备类 -->
        <h2>6. 数字设备类中的免除对象</h2>

        <h3>㈎ 一般环境（住宅、商业、轻工业区域）使用设备中的免除对象</h3>
        <h4>1)</h4>
        <ul>
            <li>手机自动触控设备</li>
            <li>电子保险箱</li>
            <li>MPPT太阳能控制器</li>
            <li>自动胶带分配器</li>
            <li>电气消磁器</li>
        </ul>

        <h4>2)</h4>
        <p>采用红外通信方式的遥控设备（例如：电视遥控器等）以及摄像机镜头</p>

        <h4>3)</h4>
        <p>无需独立电源、一对一连接的：电缆、电缆连接器、电缆附件</p>

        <h4>4)</h4>
        <p>用于简单计算或测量用途的便携式电子设备：</p>
        <ul>
            <li>便携式电子计算器</li>
            <li>数字体重秤</li>
            <li>数字温湿度计</li>
            <li>数字体温计</li>
            <li>数字血糖仪</li>
            <li>数字水平仪</li>
            <li>数字万用表</li>
            <li>或类似设备</li>
        </ul>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h4>5)</h4>
        <ul>
            <li>电打火机（用于点火用途）</li>
            <li>学校等教育机构用于科学实验的组装用品套件（套装）</li>
        </ul>
        <div class="note-box">※ 仅限采用<span class="highlight">USB或电池（包括充电电池）</span>供电运行的设备。</div>

        <h3>㈏ 使用以下电源的变速电力驱动设备：</h3>
        <ul>
            <li>高压配电网（额定电压超过1kV）</li>
            <li>特殊电源</li>
        </ul>

        <h3>㈐ 不作为通用产品使用，而是安装于特殊设备内部使用的：</h3>
        <ul>
            <li>组件</li>
            <li>部件型电源供应装置</li>
        </ul>

        <h3>㈑ 电池类：</h3>
        <ul>
            <li>备用电池</li>
            <li>电子烟电池</li>
            <li>移动电源（直流输入输出）</li>
        </ul>

        <h3>㈒ 升降压电缆类：</h3>
        <ul>
            <li>点烟器电源线</li>
            <li>直流输入输出转换器</li>
        </ul>

        <h3>㈓ 数字设备类中：</h3>
        <ul>
            <li>额定输入超过交流1,000V或直流1,500V的设备</li>
            <li>非通用机械、设备上安装的特殊结构数字设备</li>
        </ul>

        <!-- 7. 交通工具用电动设备类 -->
        <h2>7. 交通工具用电动设备类中的免除对象</h2>
        <ul>
            <li>额定输入功率超过10kW的设备。</li>
        </ul>

        <!-- 总结与解读 -->
        <h2>与农业用电动喷雾器相关的适用条款说明：</h2>
        <div class="note-box" style="border-left-color: #C00000;">
            <p><strong>根据上述 <span class="highlight-warn">第2项 ㈑ 第4条</span>：</strong></p>
            <p>“采用USB或电池（包括充电电池）供电运行的手部消毒器等电气喷雾设备、电动喷雾器、电动香氛除臭设备……”</p>
            <p><span class="highlight-warn">电池供电的农业用电动喷雾器属于免于广播通信器材适合性评价（KC EMC评价）的对象。</span></p>
            <p><strong>因此，若产品为：</strong></p>
            <ul>
                <li>农业用途；</li>
                <li>电池（充电电池）供电；</li>
                <li>无无线通信功能；</li>
                <li>属于电动喷雾器；</li>
            </ul>
            <p>则可依据该条款作为<span class="highlight-warn">“不需要进行韩国KC电磁兼容（EMC）适合性评价”</span>的依据。</p>
